## 🚀 Tech Stack

- **Framework:** Astro 6.x
- **Styling:** Tailwind CSS 3.x
- **Deployment:** Cloudflare Pages

## 🛠️ Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```
